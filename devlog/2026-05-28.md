## 架构决策

### 右下角弹出面板设计
用户明确要求"固定到电脑右下角，点击就会出来"的样式：
- 无边框 frameless 窗口，340×480
- transparent: true 实现圆角
- 失去焦点自动隐藏（blur 事件）
- alwaysOnTop 置顶显示
- skipTaskbar 不显示任务栏图标
- 系统托盘图标 + 点击切换显示/隐藏

### 玻璃质感浅蓝主题
CSS 全部使用半透明 RGBA 颜色，模拟 frosted glass 效果：
- 主体背景：rgba(215, 235, 255, 0.88)
- 卡片背景：rgba(255, 255, 255, 0.85)
- 圆角边框 + 内阴影高光
