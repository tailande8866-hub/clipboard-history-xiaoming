# 需求文档 — Windows 历史粘贴板

## 项目概述

一款运行在 Windows 上的本地历史粘贴板管理软件，自动记录用户复制的文字和图片，支持查看、搜索、管理和再次粘贴。

## 目标用户

Windows 用户，需要频繁复制粘贴文字和图片，希望能回溯历史复制内容。

## 功能需求

### F1: 自动记录粘贴板内容
- 软件后台运行时，自动检测并记录用户复制的内容
- 支持两类内容：**文字** 和 **图片**
- 连续复制相同文字时，仅更新记录时间，不产生重复条目
- 相同图片全局去重（基于 MD5），仅更新记录时间

### F2: 历史记录展示
- 以**时间降序**排列的卡片列表
- 每条记录显示为一张卡片，包含：内容预览、时间戳、操作按钮
- 置顶的记录始终排在最前面
- 文字卡片：显示文本内容（最多 5 行，超出截断）
- 图片卡片：显示缩略图（最大高度 200px）

### F3: 搜索
- 顶部搜索框，输入关键词实时过滤
- 仅搜索文字记录
- 300ms 防抖，避免频繁查询
- 清空搜索框恢复全部显示

### F4: 管理功能
- **置顶**：将记录固定到列表顶部，置顶项不被清理
- **取消置顶**：恢复普通状态
- **删除**：删除单条记录（带确认弹窗），图片记录同步删除文件
- **重新复制**：点击可将内容重新写入系统粘贴板，方便粘贴到别处

### F5: 保留期限设置
- 下拉选择：**1 天 / 3 天 / 5 天**
- 默认值：3 天
- 超过期限的非置顶记录自动清理
- 切换期限时立即执行一次清理

### F6: 系统托盘 + 快捷键
- 软件常驻系统托盘（右下角图标）
- 全局快捷键 **Ctrl+Shift+V** 呼出/隐藏主界面
- 托盘右键菜单：显示主界面 / 退出
- 关闭窗口时隐藏到托盘，不退出程序

### F7: 开机自启
- 安装后默认开机自动启动
- 启动时最小化到托盘，不弹出主窗口

## 非功能需求

### NF1: UI 设计
- 简洁直观，卡片式布局
- 主色调：**淡蓝色**
- 操作按钮有 hover 效果
- 窗口大小适中，可调整
- 无历史时显示空状态提示

### NF2: 性能
- 粘贴板轮询不影响系统性能
- 列表支持大量记录（虚拟滚动或分页）
- 图片缩略图延迟加载

### NF3: 存储
- 所有数据存储在本地（`%APPDATA%/clipboard-history/`）
- 不上传任何数据到网络
- 图片存文件系统，数据库只存元数据

### NF4: 安全
- 不需要网络权限
- 不读取用户其他文件
- 安装和卸载干净，无残留
